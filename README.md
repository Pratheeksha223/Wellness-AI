# WellnessAI

## AI-Powered Health, Fitness & Wellness Assistant

### Project Overview

WellnessAI is an AI-powered health and wellness assistant that helps users improve their lifestyle through personalized fitness, nutrition, hydration, and sleep recommendations.

The project uses IBM Granite Models and IBM BOB concepts to demonstrate how Artificial Intelligence can support preventive healthcare and promote healthier habits.

### SDG Alignment

**SDG 3: Good Health and Well-Being**

### Key Features

* Personalized Workout Plans
* Nutrition Recommendations
* Hydration Tracking
* Sleep Guidance
* Wellness Score Analysis
* Progress Tracking
* AI-Powered Health Insights

### AI Technologies Used

* IBM Granite Models
* IBM BOB
* Conversational AI
* Prompt Engineering
* Personalized Recommendation Engine
* Wellness Analytics
